# mean-variance-standard-deviation-calculator

## Overview

This project was completed as part of the FreeCodeCamp Data Analysis with Python Course.

## Objective

To calculate statistical measures including mean, variance, standard deviation, maximum, minimum, and sum for a 3×3 matrix using NumPy.

## Technologies Used

- Python
- NumPy

## Skills Learned

- Arrays
- Reshape, Axis Operation
- Statical Operations
